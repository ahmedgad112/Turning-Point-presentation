About the Project

Key pillars that summarize the idea and objectives of the platform

Zero Down Network (ZDN)

Automating paths and securing them through a diamond mesh structure that connects Cisco and Arista devices. This ensures that data keeps flowing through OSPF and HSRP protocols without dropping a single packet, even if an entire router fails.

Zero Down Service (ZDS)

Building an integrated computing environment (HCI) based on a Proxmox VE stack to gather and manage resources. The platform allows efficient load distribution and automatically restarts virtual servers in fractions of a second as soon as any physical hardware failure is detected.

Ceph Distributed Storage

Building an integrated computing environment (HCI) based on a Proxmox VE stack to gather and manage resources. The platform allows efficient load distribution and automatically restarts virtual servers in fractions of a second as soon as any physical hardware failure is detected.

Live Migration

Applying the 'live migration' technology to move servers and running systems between physical devices in real-time. The processor and RAM records are transferred over a 10-gigabit Arista network, allowing hardware maintenance without the end user even noticing any downtime.

Seamless Roaming (BATMAN-adv)

Developing a wireless mesh network managed by the BATMAN-adv protocol, which runs inside the system kernel at layer two. This innovation allows mobile devices and VoIP calls to switch between access points (APs) completely smoothly and in less than $50 ext{ms}$ without any signal loss.

Sovereign Cloud Platform

A smart control panel fully developed locally using Laravel and Tailwind CSS, directly connected to the Proxmox API. It gives users an experience similar to global hosting companies (like Hostinger) for creating servers and managing email and domain naming (DNS) automatically, with 100% complete digital sovereignty.
